A versão 4.0.0 do LAND IT encontra-se agora disponível. Segue-se a lista de alterações e novas funcionalidades.

<!-- truncate -->

-    Alteração da framework para visualização de mapa e geometrias: removeu-se totalmente o ArcGIS JS e agora o LAND IT utiliza o OpenLayers
-    Incorporação de camadas raster no sistema:
     -  O utilizador pode importar camadas raster no formato TIFF ou adicioná-las através de um URL nos formatos TIFF e WMS
     -  Adição de um pequeno menu para alterar a camada de fundo do mapa, onde existem tanto camadas vetorais como raster
-    Melhorias na lengenda: a legenda agora é interativa, i.e., o utilizador pode selecionar elementos da legenda e as geometrias correspondentes aparecerão destacadas no mapa
-    Adição de uma lista de atributos: para cada camada é possível abrir a lista e analisar todos os atributos e respetivos valores das geometrias contidas nesta
-    Adição de testes unitários para garantir a constante integridade e correto funcionamento do sistema
-    Expansão do sistema de atualização de camadas: todas as camadas de AIGP e territoriais agora pertencem a este sistema e quando atualizadas é necessário propagar as alterações para as versões dos cenários afetadas
-    Correções à funcionalidade de atualizar a POSA
-    Correções de bugs detetados pelos testes unitários
-    Melhorias de performance ao guardar geometrias na base de dados
