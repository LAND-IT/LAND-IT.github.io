A versão 3.9.0 do LAND IT encontra-se agora disponível. Segue-se a lista de alterações e novas funcionalidades.

<!-- truncate -->

-    Foram realizadas atualizações ao método de cálculo dos serviços de ecossistemas: para as AIGP mais recentes alguns tipos de FGC não contam para a atribuição de EP;
-    Agora as entidades e as variantes de execução estão também presentes na POSP e não só nas transformações;
-    A funcionalidade de importação de cenários agora aceita que seja possível importar só a POSP, sem as UIs;
-    Agora é possível atribuir automaticamente transformações a polígonos sem transformações com base na base de conhecimento sem estes terem uma transformação anterior;
-    As atualizações à base de conhecimento podem devolver conflitos que o utilizador terá de resolver, nomeadamente ao importar um cenário;
-    NOVA FUNCIONALIDADE: Agora é possível atribuir transformações a grupos de polígonos que tenham as mesmas características do território;
-    No menu de transformações, nas unidades de transformação, ao editar uma unidade sem filtro de versão o utilizador estará a atualizar uma unidade que afetará todos as versões em que ela está presente. Esta ação tem um aviso de confirmação. Para alterar uma unidade e estas alterações serem propagadas só para uma versão basta selecionar uma versão no respetivo filtro;
-    Ao adicionar uma camada edição o utilizador agora pode eliminar os atributos que definiu.
-    Correção de bugs:
     -    Algumas camadas com polígonos sobrepostos estavam a dar erro ao serem importadas para o sistema;
     -    Camadas com caracteres especiais no nome estavam a dar erro ao serem exportadas;
     -    Correção de alguns valores no menu das estatísticas de uma versão;
     -    Ao adionar/eliminar camadas territoriais/AIGP a lista de camadas não estava a ser atulizada automaticamente.
