A versão 4.7.0 do LAND IT encontra-se agora disponível. Segue-se a lista de alterações e novas funcionalidades.

<!-- truncate -->

-    Adição de um novo mecanismo para cálculo automático das vedações (ainda se encontra numa fase inicial).
-    Adição do conceito de atributos estendidos.
     -   Agora os polígonos de transformação podem ter uma coleção de atributos extra, como a especificação das vedações que incluem, o ID da UI definido inicialmente e o protocolo associado à Rede Primária (PRA, CP21,...)
-    Deslocação do LAND IT para um novo host.
-    Correção dos investimentos imateriais, apresentando apenas as opções com os códigos Or47 e Or48.
-    Nova funcionalidade de exportação de uma versão com os filtros aplicados.
-    Adição da possibilidade de correção de intersecções ao copiar um polígono de uma camada de utilizador para a POSP.
-    Ajuste à operação buffer nas camadas de utilizador, de forma a permitir a sua aplicação a várias linhas em simultâneo.
-    Correção de bugs ao importar OIGP para o LAND IT.
-    Correção de bugs associados à importação de ficheiros.
