A versão 4.1.0 do LAND IT encontra-se agora disponível. Segue-se a lista de alterações e novas funcionalidades.

<!-- truncate -->

-    Inserção de ferramentas para edição de geometrias:
     -   União
     -   Traçar
     -   Identidade
     -   Recortar
     -   Divisão
-    Agora é possível alterar o nome de uma versão.
-    Adição de um logótipo temporário.
-    As queries a camadas de edição estão novamente a funcionar.
-    Correções de bugs de renderização de polígonos no mapa.
-    Correção de bugs nas operações complementares.
-    Correções nos testes unitários.
-    Otimizações no método de importação de cenários.
-    Melhorias de performance.

<details>
<summary>Mostrar detalhes</summary>

-    Detalhes das novas ferramentas de edição de geometrias:
     -   União: Esta ferramenta permite combinar diferentes polígonos num único novo polígono. O utilizador pode determinar qual o polígono cujos atributos serão preservados no resultado final.
     -   Traçar: Permite criar novos polígonos ao seguir o contorno de geometrias já existentes, proporcionando uma forma mais precisa e eficiente de desenhar novas formas.
     -   Identidade: Permite dividir polígonos existentes com base num polígono desenhado, criando novos polígonos correspondentes às interseções.
     -   Recortar: Esta ferramenta permite extrair uma parte de um polígono que esteja contida dentro de um polígono desenhado pelo utilizador. A restante parte do polígono original é eliminada.
     -   Divisão: Permite separar polígonos em partes distintas ao desenhar uma linha sobre eles.
-    As melhorias de performance foram possíveis graças a duas correções:
     -   O GDAL, utilizado em operações entre camadas, agora utiliza ficheiros GPKG em vez de shapefiles;
     -   Não são permitidas operações entre camadas com polígonos que tenham mais de 700 pontos, caso tal aconteça estes são divididos em polígonos menores.
