A versão 5.0.0 do LAND IT encontra-se agora disponível. Segue-se a lista de alterações e novas funcionalidades.

<!-- truncate -->

-    Criação de um novo menu no LAND IT destinado a dados correspondentes a todo o município de Mação, como o cadastro, o declive, a adesão e os contratos;
-    Vários componentes foram adicionados a docks, tal como o mapa e a tabela de atributos, de modo a que o utilizador possa mover e redimensionar estes componentes como lhe for preferível;
-    Reconstrução do modo de adição/importação de camadas de informação: a estrutura das camadas é definida num ficheiro JSON que será importado para o LAND IT
-    Ao adicionar uma camada de informação é possível definir relações entre a camada a adicionar e as camadas existentes. Tal como relacionar o ID de uma propriedade no ficheiro da adesão com a camada do cadastro.
-    Implementação de mecanismos de validação dos dados a serem inseridos nas camadas de informação, tais como validações de email, datas, nacionalidade e nº de telefone.
-    Adição de camadas vista - camadas que permitem visualizar um conjunto de camadas que se relacionam entre si, na prática executam junções entre as camadas selecionadas e numa mesma linha mostram atributos das diversas camadas.
-    Implementação das funcionalidades de adição, remoção e edição de linhas nas camadas de informação.
-    Criação de um sistema de notificações em tempo real: agora os utilizadores são avisados quando novas camadas com informações das AIGP ou do território são adicionadas ou eliminadas.
-    Adição de um mecanismo de notificações para quando um cenário é partilhado com outros utilizadores.
-    Implementação de colaboração de edição em tempo real de desenho de polígonos, para que vários utilizadores possam trabalhar num mesmo cenário em simultâneo. Criação de um sistema de cores para cada sessão, permitindo identificar cada utilizador e quem está a desenhar.
-    Edição das camadas POSA manualmente
     -   adicionar e remover polígonos
     -   editar atributos
     -   aplicar ferramentas de edição de geometrias
-    Nova interface de login para uma experiência mais intuitiva.
-    Otimizações na interface do LAND IT para um carregamento e renderização mais rápidos.
-    Correção de diversos bugs.
