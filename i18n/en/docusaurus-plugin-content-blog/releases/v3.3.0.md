LAND IT version 3.3.0 is now available. Below is the list of changes and new features.

<!-- truncate -->

-    Export of scenarios with shapefiles and Excel files
-    Improvements in scenario import
-    Addition of intangible investments
-    Added the possibility to exclude polygons from certain complementary operations
-    Update of the calculation of transformation unit costs
-    Update of some layer types
-    Addition of total units per transformation unit (total m3 in a transformation)
-    Mosaics are now considered as a determining factor in the creation of a transformation unit
-    Bug fixes
