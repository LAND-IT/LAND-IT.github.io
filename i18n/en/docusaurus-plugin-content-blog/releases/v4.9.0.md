LAND IT version 4.9.0 is now available. Below is the list of changes and new features.

<!-- truncate -->

-    Creation of the concept of information layers to store non-geographic data, such as owners.
-    Several functionalities can now be applied to more than one polygon simultaneously.
-    Improvements in the clip tool: it is now possible to choose between the intersection of geometries or their subtraction.
-    Improvement when adding polygons to the POSP: it is possible to group polygons to define POSP attributes, decreasing the number of fields the user has to fill.
-    Bug fixes: legend, define transformations and copy geometries to the POSP of territory layers.
