LAND IT version 3.4.0 is now available. Below is the list of changes and new features.

<!-- truncate -->

-    Change to user permissions
-    Added logout functionality
-    Correction to the measurement widget
-    Addition of the ecosystem services page in the Transformation Editing menu
-    Addition of a button to zoom to selected geometries or to all displayed geometries
-    Addition of the details button in each transformation (Transformation Editing menu) where the polygons of a transformation can be viewed
-    Bug fixes
