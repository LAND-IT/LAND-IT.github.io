Version 5.11.0 of LAND IT is now available. Below is the list of changes and new features.

<!-- truncate -->

- Scenario Sharing now opens automatically with the current scenario already selected.
- When importing a Shapefile, there is now the option to use the Shapefile name as the layer name.
- Ability to hide rows in the attribute table.
- Fixed the interface of the intersection engine (map legend and new icons to differentiate the type of correction).
- The background map in the Scenarios Editing now remembers the user's selection.
- Collaboration:
    - Added the ability to create private or shared discussions with selected users.
    - Improved writing of long messages in discussions.
    - Added collaborative annotations on the map.
- Added functionality allowing users to enable/disable extensions.
- Added documentation about Discussions and Extensions.
- Correction of various bugs.
