LAND IT version 5.1.0 is now available. Below is the list of changes and new features.

<!-- truncate -->

-    New feature: adding data to information layers via a CSV
-    New feature: when editing/adding a row to an information layer it is possible to obtain the filling of some fields based on alternative fields - such as, for example, the codes (NIC and PRD) of the cadastre
-    Implementation of read and write permissions when sharing a scenario
-    Addition of new interaction mechanisms between the attribute table and geometries on the map
-    Implementation of a new method to render layers with thousands of geometries
-    Bug fixes related to permissions
