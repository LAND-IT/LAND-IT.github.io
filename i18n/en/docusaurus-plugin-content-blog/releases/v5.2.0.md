LAND IT version 5.2.0 is now available. Below is the list of changes and new features.

<!-- truncate -->

-    New feature: addition of Portuguese address validation in information layers
-    New feature: tool that allows copying polygons to the POSA
-    Addition of "serial" type attributes to information layers, which allows each row of one of these layers to have a unique ID assigned automatically
-    Improvements in the intersection mechanism and correction of overlaps
-    Updates to the POSP polygon insertion and editing menu
-    Improvements in the filter mechanism
-    Correction of various bugs
