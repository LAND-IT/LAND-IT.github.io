LAND IT version 5.4.0 is now available. Below is the list of changes and new features.

<!-- truncate -->

-    Performance optimizations
-    Optimizations in the functionality of copying geometries between layers and versions
-    Addition of a version mechanism for AIGP layers and territory layers
-    Possibility to change the POSA and save changes without propagating them to the version
-    Addition of the translation mechanism - small parts of LAND IT can already be viewed in English
-    Addition of the possibility to change the background map in all auxiliary maps
-    All collaborative editing features have been revised and improved to provide a superior user experience.
-    Implementation of a notification system, accessible through the upper right corner of the interface.
-    Bug fixes
