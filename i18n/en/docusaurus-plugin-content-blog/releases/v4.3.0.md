LAND IT version 4.3.0 is now available. Below is the list of changes and new features.

<!-- truncate -->

-    New tool to edit layer styles (1st version).
-    New format for the tools panel, simpler and more intuitive.
-    Improvements in the geometry selection method, to be more similar to what is found in main GIS applications.
-    Update of the undo/redo buttons design.
-    Corrections to geometric tools (undo and redo).
-    Corrections to filters and addition of new attributes.
