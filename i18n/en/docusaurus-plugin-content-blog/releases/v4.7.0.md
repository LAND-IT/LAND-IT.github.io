LAND IT version 4.7.0 is now available. Below is the list of changes and new features.

<!-- truncate -->

-    Addition of a new mechanism for automatic calculation of fences (still in an initial phase).
-    Addition of the concept of extended attributes.
     -   Now transformation polygons can have a collection of extra attributes, such as the specification of the fences they include, the UI ID initially defined and the protocol associated with the Primary Network (PRA, CP21,...)
-    Relocation of LAND IT to a new host.
-    Correction of intangible investments, presenting only options with codes Or47 and Or48.
-    New functionality to export a version with applied filters.
-    Added the possibility of correcting intersections when copying a polygon from a user layer to the POSP.
-    Adjustment to the buffer operation in user layers, to allow its application to several lines simultaneously.
-    Correction of bugs when importing OIGP into LAND IT.
-    Correction of bugs associated with file import.
