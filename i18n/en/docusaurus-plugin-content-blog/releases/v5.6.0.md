Version 5.5.0 of LAND IT is now available. Below is the list of changes and new features.

<!-- truncate -->

- Update of the COS classes to the latest version (2023 v1). 
- Addition of more COS versions to LAND IT. Versions 2018 v2, 2018 v3, and 2023 v1 are now available. 
It is also possible to change the COS version of a POSA, as well as to import a scenario that uses an older COS version. 
When exporting, the user can choose which COS version they want to obtain. 
- Adjustments to the maximum zoom level allowed on the map to improve the user experience. 
- Warning pop-ups now have a different style to make them more visible. 
- Correction of various bugs.
