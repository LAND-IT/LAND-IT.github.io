# Interface

No centro temos o mapa, onde são visualizados os polígonos e toda a informação espacial.

Do lado esquerdo está a área das camadas, onde é feita a gestão completa das camadas — ativar, desativar, reorganizar e consultar as informações associadas.

E do lado direito encontram-se as ferramentas geométricas, que permitem trabalhar diretamente sobre os polígonos: selecionar, editar, desenhar ou aplicar operações mais avançadas.
