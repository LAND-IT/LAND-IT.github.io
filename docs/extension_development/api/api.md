# APIs

No LAND IT, existem duas APIs definidas que permitem que os módulos de extensão interajam com a aplicação principal do LAND IT e os seus dados: o **LAND IT Back-end Service API** (para módulos de back-end) e o **LAND IT Front-end Extensions SDK** (para módulos de front-end).
