# Tipos de Módulos

Uma extensão é composta por múltiplos **módulos**, que podem ser de diversos tipos, conforme o seu propósito e domínio de execução (back-end/front-end). Esta secção explica os diferentes tipos de módulos de extensão suportados no LAND IT.
